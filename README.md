# Number-Guesser
A simple game that involves guessing a random number. There are 3 difficulties.

### Example:

```
-Number Guesser-

Easy = E | Medium = M | Hard = H
Choose difficulty: M
Guess my number between 1-10
You have 3 Guesses!
Guess: 5
Lower
Guess: 3
Higher
Guess: 4
The number was  4
You guessed my number in 3 tries on Medium difficulty!
```
